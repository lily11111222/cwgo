# CWgo——city walk记录App

## 功能
1. 搜索、导航
2. 记录city walk路线
3. 动态发帖（包括图片选择、裁剪、帖子带有路线记录）
4. 浏览帖子（包括自己的和所有人的、下拉刷新）
5. 注册登录（包括邮箱验证）
6. 修改个人信息（用户名、个性签名、头像等）

## 开发环境和框架
### 客户端：Android studio
### 服务器端：IDEA，Springboot框架、Mybatis框架、MySql数据库
### 阿里云服务器
